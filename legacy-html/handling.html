⚙️ 1. Setup & Installations

Open your terminal and run these commands to install React Router and Tailwind CSS:

cd my-store
npm install

# 2. Install React Router DOM
npm install react-router-dom

📁 2. Project Structure

Create the following folders and files inside your src directory before we start coding:

my-store/
└── src/
    ├── App.jsx ← (Main file with Router & State)
    │

        ├── TShirtPage.jsx ← (Page 1: Example)
        └── SneakerPage.jsx ← (Page 2: Your Challenge)

🌐 3. The App Router

In this lesson, we will keep things simple. App.jsx will only handle routing between our pages. No State or Props are needed here!

📁 src / App.jsx
import React from 'react';
import { BrowserRouter as Router, Routes, Route } from 'react-router-dom';
import TShirtPage from './pages/TShirtPage';
import SneakerPage from './pages/SneakerPage';

export default function App() {
  return (
    <Router>
      <Routes>
        <Route path="/" element={<TShirtPage />} />
        <Route path="/sneaker" element={<SneakerPage />} />
      </Routes>
    </Router>
  );
}
